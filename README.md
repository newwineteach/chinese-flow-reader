# 華語流利讀 Chinese Flow Reader

智慧華語發音練習與比對工具

## 🌐 線上使用

👉 **立即使用：** https://newwineteach.github.io/chinese-reader/

## ✨ 功能特色

- 🎯 **智慧斷詞顯示** - 自動將課文分詞，方便閱讀
- 🎵 **AI 標準發音導讀** - Web Speech API 提供標準華語發音
- 🎤 **語音錄製與辨識** - 錄下你的發音並自動辨識
- 📊 **雙波形視覺化比對** - 綠色波形（AI）vs 橘色波形（你）
- 🔄 **AI vs 使用者對比播放** - 先播 AI，停1秒，再播你的錄音
- 💯 **準確度評分系統** - 自動分析發音準確度（0-100%）
- 💾 **錄音下載功能** - 保存練習記錄

## 🎓 適用對象

- 👨‍🎓 華語學習者（初級~高級）
- 👩‍🏫 華語教師
- 📚 華語自學者
- 🌍 海外華語學習社群

## 💻 使用說明

### 步驟 1：選擇課文
在左側面板選擇課文（L1-L10）或輸入自己的文字

### 步驟 2：聆聽 AI 發音
點擊「**播放導讀**」按鈕，聽 AI 如何發音

### 步驟 3：錄製你的發音
1. 點擊「**口說練習**」按鈕
2. 允許使用麥克風
3. 對著麥克風清楚朗讀
4. 再次點擊「**停止錄音**」

### 步驟 4：比對發音
錄音完成後，頁面底部會出現比對面板：
- 📊 查看雙波形比對圖
- 🔊 使用「**對比播放**」聽差異
- 💯 查看準確度評分和回饋
- 💾 下載你的錄音

## 🎨 介面預覽

- 右側：課文顯示、播放控制、錄音按鈕

- 📝 語音辨識結果

## 🔧 技術棧

- **前端框架**: React 18 (CDN)
- **樣式**: Tailwind CSS (CDN)
- **語音合成**: Web Speech Synthesis API
- **語音辨識**: Web Speech Recognition API
- **音訊處理**: Web Audio API, MediaRecorder API
- **視覺化**: HTML5 Canvas
